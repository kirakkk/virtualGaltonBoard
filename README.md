# 教学型虚拟高尔顿板 (Virtual Galton Board for Teaching)

这是一个基于 Web 的教学推演工具，通过逼真的物理模拟直观地展示了抛物实验、二项分布（Binomial Distribution）以及正态分布近似（Normal Approximation）的历史经典实验——高尔顿板（Galton Board）。

## ✨ 核心特色

*   **真正的物理模拟与统计分离**：不同于简单的动画演示，本项目采用了严格的物理引擎（Matter.js）进行刚体碰撞模拟。同时，在统计口径上采用“过线即记样”，避免了底部小球堆积造成的二次碰撞干扰，使统计数据更贴近真实的二项分布理论值。
*   **交互式教学面板**：实时展示当前实验的各项理论与观测数据：
    *   动态更新样本总数 ($m$)
    *   实时计算理论均值 ($\mu = np$) 与观测均值
    *   实时计算理论标准差 ($\sigma = \sqrt{npq}$) 与观测标准差
    *   量化展示实验数据与理论分布的偏差幅度
*   **高度可调的实验参数**：
    *   **层数调整**：支持动态调节钉阵层数（$n = 6 \sim 18$），直观体会层数增加时二项分布向正态分布的演化。
    *   **速率控制**：可调节小球自动投放的速率（$1 \sim 10$ 球/秒），既能做缓慢的单次推演，也能进行高密度的极限测试。
    *   **模拟模式**：支持单球演示、100球批量实验、500球批量实验。
*   **丰富的可视化元素**：
    *   动态直方图统计（蓝色柱体代表观测频数）
    *   二项分布理论曲线（绿色折线）
    *   正态分布近似曲线（黄色虚线）
    *   单个小球下落轨迹追踪

## 🛠 技术栈

*   **前端结构**：原生 HTML5 + JavaScript (ES6)
*   **物理引擎**：[Matter.js](https://brm.io/matter-js/) (0.19.0) - 处理复杂的多体物理碰撞与动力学计算。
*   **样式构建**：[Tailwind CSS](https://tailwindcss.com/) (CDN) - 提供现代化的深色模式 UI (Dark Mode) 和响应式布局。
*   **动态渲染**：HTML5 `<canvas>` - 利用自定义 HUD 绘制统计直方图与高斯曲线。

## 🚀 如何运行

本项目是一个纯前端的单文件应用，零构建依赖，开箱即用：

1. 克隆本项目到本地：
   ```bash
   git clone https://github.com/kirakkk/virtualGaltonBoard.git
   ```
2. 直接在现代浏览器中双击打开 `galton_board_stable_teaching_v2.html` 即可运行。
3. （可选）如果你想部署在本地服务器上，可以使用任何轻量级的静态服务器，例如：
   ```bash
   npx http-server . -p 8080
   ```
   然后访问 `http://localhost:8080/galton_board_stable_teaching_v2.html`

## 📚 教学意义背景

高尔顿板是由英国生物学家、统计学家弗朗西斯·高尔顿（Francis Galton）发明的装置。当小球从顶部落下，每次撞击钉子时，小球都会以约 50% ($p=0.5$) 的概率向左或向右弹跳。

如果装置有 $n$ 层钉子，小球向右弹跳的次数 $X$ 服从二项分布 $X \sim B(n, 0.5)$。当 $n$ 足够大时，由**棣莫弗-拉普拉斯中心极限定理**可知，二项分布将趋近于正态分布 $N(n/2, n/4)$。本项目完美利用数字形式重现了这一优雅的数学与物理融合的过程。
